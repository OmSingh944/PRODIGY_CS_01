# PRODIGY_CS_01 - Caesar Cipher Tool

## Objective
This project implements the Caesar Cipher algorithm in Python for encryption and decryption of messages.

## Features
- Encrypt messages
- Decrypt messages
- Brute Force Attack
- Save output to text file

## Technologies Used
- Python

## How to Run
1. Run the Python file.
2. Choose an option from the menu.
3. Enter the message and shift value.
4. View the result.
